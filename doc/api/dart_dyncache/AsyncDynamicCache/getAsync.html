Future<V?> getAsync(
  1. dynamic key
)

If key is known to be associated with a future value, this method waits until that future is completed.

If key is an auxiliary key and the associated AuxiliaryKeyManager.valueNeeded is true, or if setAsync was called with an auxiliary key who's associated AuxiliaryKeyManager.valueNeeded is true this method will miss even if the future is known.

Implementation

Future<V?> getAsync(dynamic key) async {
  if (_waitingOnMap.containsKey(key)) {
    await _waitingOnMap[key];
  }
  return get(key, throwOnUnsafe: false);
}
