dart_dyncache

dart_dyncache library Null safety

An attempt at an all-purpose dynamic caching library.

Classes

AsyncBasicDynamicCache<K, V>
A wrapper for a BaseDynamicCache that implements DynamicCache and natively supports Future values.
AuxiliaryKeyManager<KA, KO, V>
A class used to manage conversions between a main key of type KO and an auxiliary key of type KA.
BaseDynamicCache<K, V>
An abstract cache class with a weight function that supports eviction and auxiliary keys.
DynamicCache<K, V>
Abstract class for the basic functionality of a cache.
FIFODynamicCache<K, V>
A Dynamic cache using a First In First Out implementation.
LFUDynamicCache<K, V>
A Dynamic cache using a Least Frequently Used implementation.
LRUDynamicCache<K, V>
A Dynamic cache using a Least Recently Used implementation.
OrderedMap<K, V>
An implementation of OrderedStorage using a sorted array and a key-to-index map.
OrderedStorage<K, V>
A class that keeps a weighted K-V pairing.
SimpleDynamicCache<K, V>
A basic implementation of BaseDynamicCache.

Exceptions / Errors

DynCacheAuxKeyCollisionException
Thrown if there is a key collision with auxiliary keys and BaseDynamicCache.checkAuxiliaryKeys is true.
DynCacheException
DynCacheTypeException
Thrown when a BaseDynamicCache cannot handle a key type.
DynCacheUnsafeException
Thrown on unsafe BaseDynamicCache.get behavior.
